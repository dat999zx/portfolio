# Portfolio học phần

Bộ file portfolio gồm:
- `index.html` — Giới thiệu
- `du-an.html` — Dự án / bài tập LMS
- `tong-ket.html` — Tổng kết khóa học
- `styles.css` — Giao diện và animation
- `script.js` — Hiệu ứng xuất hiện khi cuộn

## Cách dùng trên GitHub Pages
1. Upload toàn bộ file vào cùng một repo.
2. Vào **Settings > Pages**.
3. Chọn **Deploy from a branch**.
4. Chọn branch `main` và folder `/root`.
5. Mở link GitHub Pages của repo.

## Tùy chỉnh
Hãy sửa lại các thông tin cá nhân trong `index.html`:
- Họ và tên
- Mã sinh viên
- Trường
- Email
